
# xor

  tiny xor utility function

## Installation

  Install with [component(1)](http://component.io):

    $ component install component/xor

## API

### xor(bool, bool)

XOR two boolean values.

```js
xor(true, true) // false
xor(true, false) // true
xor(false, true) // true
xor(false, false) // false
```
